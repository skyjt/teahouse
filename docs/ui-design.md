## 1. 设计基调

**参考微信桌面版的设计语言**：三栏结构、大量留白、极轻的分割线、克制的色彩。"简约优雅、克制但不老气"落到具体手段上：

- **一个主题色**点缀全局（按钮、选中态、自己的气泡底色），其余全部中性灰阶；主题色定为**茶青 `#3D8B6B`**（决议 #1）。
- **系统字体栈**：`PingFang SC / Microsoft YaHei / Noto Sans CJK`，正文 14px，绝不用衬线、不用艺术字。
- **图标统一为项目内自绘线性 SVG**（1.6px 描边、圆角端点、继承当前文字色），不混用拟物/彩色图标，不用 emoji 充当系统图标；设置图标必须画成清晰齿轮，不使用放射状“太阳”隐喻。emoji 仅保留在“可发送的表情字符”和头像模板场景。
- **动效克制**：150–200ms 缓动，只用于面板切换与气泡入场，不做花哨过场。
- 圆角小而统一（4px / 8px 两档），阴影只给浮层。
- 深色主题 v0.4（roadmap），实现时颜色全部走 token 变量，先把底子留好。

## 2. 主窗口：三栏结构

```
┌──────┬──────────────────────┬────────────────────────────────────────┐
│  我  │ [搜索]        [+]   │ 张三   ● 在线 · 某某科技/研发部/后端组  ⋯ │
│      ├──────────────────────┼────────────────────────────────────────┤
│ 聊天 │ ● 张三        10:24 │              ┌──────────────────────┐   │
│ 聊天 │   好的，收到    (2)  │              │ 需求文档发我一下？     │   │
│      │ 讨论组          9:15 │              └──────────────────────┘   │
│ 通讯 │   李四: [图片]       │   ┌────────────────────┐               │
│ 通讯 │ ○ 王五        昨天  │   │ [文件] 需求文档v3.docx │            │
│ 录   │   [文件] 设计稿.zip  │   │ 2.4 MB   已送达      │               │
│      │                      │   └────────────────────┘               │
│      │                      ├────────────────────────────────────────┤
│ 设置 │                      │  表情 截图 图片 文件                     │
│      │                      │  输入消息…                              │
└──────┴──────────────────────┴────────────────────────────────────────┘
  56px      250px（可拖拽）              自适应，≥560px
```

| 栏 | 内容 | 规格 |
|---|---|---|
| ① 导航栏 | 上：我的头像（点击=编辑资料）→ 聊天 → 通讯录；下：设置 | 64px 固定；深色外侧 + 浅灰圆角胶囊；图标带未读总角标 |
| ② 列表栏 | 随导航切换：会话列表 / 通讯录；顶部搜索框常驻 | 250px，可拖 200–320px |
| ③ 内容区 | 聊天界面 / 联系人资料卡 / 空状态 | ≥560px；未选会话时显示淡灰 logo 空状态 |

- 窗口最小 960×640，记忆上次大小与位置；关闭按钮默认最小化到托盘（设置可改）。

## 3. 列表栏 · 会话列表（导航=聊天）

- 条目：头像（带在线小圆点）、名称、最后消息摘要、时间、未读角标；免打扰会话角标改为静默小灰点。
- 摘要规则：`李四: [图片]`、`[文件] 设计稿.zip`、草稿优先显示红字 `[草稿] …`。
- 排序：置顶在前，其余按最后消息时间倒序（微信同款）。
- 右键菜单：置顶 / 免打扰 / 标为已读 / 从列表移除（不删历史，下次来消息会回来）。
- 顶部 `[+]`：发起讨论组（多选联系人，见 §7.1）。

## 4. 列表栏 · 通讯录（导航=通讯录）

```
[搜索]
▾ 讨论组
   # 产品讨论组 (8人)
▾ 某某科技 (23/41 在线)
   ▾ 研发部 (12/18)
      ▾ 后端组 (5/7)
         ● 张三  Windows  ← 在线
         ● 李四  macOS
         ○ 王五 · 离线     ← 灰显 + "离线"副文字，沉底
      ▸ 前端组 (4/6)
   ▸ 设计部 (3/8)
▸ 未分组 (2/5)
```

- **三级树：公司 ▸ 部门 ▸ 团队 ▸ 成员**，逐级折叠，每级标题显示（在线/总数）；资料字段空缺的层级自动跳过，全空归入"未分组"。
- 组内排序：在线优先，再按拼音；离线者灰显**且昵称旁加"离线"副文字**（不只靠颜色区分，决议 #17/补充轮）。
- **历史联系人持久保留**：凡是出现过的节点永久留在通讯录（离线灰显），重启不丢。不提供手动删除（对方还在网内时删了也会回来）；"清理 90 天未上线"放设置-高级。
- **资料自动刷新（防"机器换人"）**：对方上线时、改资料时即时更新；在线期间心跳每 30s 携带资料版本号，发现版本变化立刻拉取最新资料——机器没换、用的人换了昵称/部门，最迟半分钟内通讯录就纠正过来。
- 点击成员 → 内容区显示**资料卡**：头像、昵称、公司/部门/团队、IP、平台、在线状态 + 「发消息」按钮（微信式，进入会话）；资料卡可**设置备注名**（仅本机生效，列表与搜索优先按备注显示，决议 #22）。
- 顶部固定「讨论组」分组，列出已加入的组。

## 5. 内容区 · 聊天界面

**头部**：对方昵称 + 在线状态（或组名 + 人数）。**打开会话即触发一次探活**，约 2s 内以真实结果刷新状态，离线则显示灰字"离线"——防止对着"假在线"的人发消息（F-DISC-8）。右侧 `⋯` 打开会话设置（置顶/免打扰/查看成员/清空本会话记录）。群聊点人数 → 右侧滑出成员面板（含「+ 添加成员」）。

**消息区**：

- 气泡：对方靠左白底，自己靠右主题色浅底（比微信的纯色绿更轻，显高级）；圆角 8px。
- 时间分隔：间隔 >5 分钟插一条灰色小字时间；跨天插日期。
- 文本消息：链接自动识别可点；仅气泡本身可右键=复制 / **转发**（选目标会话，P1）/ 撤回（自己的、2 分钟内，P1），消息行空白区域不触发菜单；右键菜单靠近窗口边缘时自动避让，不被裁切。
- 图片消息：等比缩略图（最大 280×280）内嵌显示，点击进大图查看器（缩放/另存为）；右键菜单含**「转发」/「添加到表情」**。
- 表情包消息：固定 120px 高度渲染（不随原图大小变化），右键可「转发」或「添加到表情」收藏对方发的表情。
- **文件消息卡片**（微信式）：

```
┌─────────────────────────┐      状态流转：
│ [文件图标] 设计稿.zip    │      等待对方接收 → 传输中(进度条+速率+取消)
│          48.2 MB         │      → 已完成(打开 / 打开所在文件夹)
│          6.2 MB/s        │      → 已取消 / 失败(重试)
└─────────────────────────┘
```

- 发送状态：发送中=自绘小转圈 → 已送达=自绘对勾；失败=红色 ! 点击重发；**对方离线=自绘时钟图标**，气泡下灰字"对方上线后自动送达"。
- 系统提示行（居中灰小字）：撤回提示、群成员变更等，尽量少，不刷屏。消息撤回后原气泡从本地消息流中隐藏，只保留"你撤回了一条消息 / 对方撤回了一条消息"提示。

**输入区**：工具条（表情 / 截图 / 图片 / 文件 / 文件夹，全部用自绘线性 SVG）+ 多行输入框 + 发送。工具条图标悬停约半秒后以淡入浮层显示中文名称，禁用态也保留名称提示。

- 表情面板双页签：**emoji** / **我的表情包**。表情包来源：聊天里任意图片右键「添加到表情」，**保存时自动压缩**（静图最长边 512px 转 WebP，GIF 动图 ≤2MB 原样保留）；面板内右键删除、拖拽排序；点击即发送。
- Enter 发送、Ctrl/Cmd+Enter 换行（设置可互换）。
- 群聊中输入 `@` 弹出成员选择浮层，被 @ 者收到加强通知（会话列表标"[有人@我]"红字，P1，决议 #22）。
- 粘贴图片 → 输入框出现预览块再发送；**拖文件/文件夹到聊天区任意位置即发送**（拖入时全区高亮蒙层提示）。

## 6. 全局搜索（微信式）

两个列表页顶部的搜索框是同一套组件，聚焦后弹出结果浮层，分类展示：

| 分类 | 范围 | 说明 |
|---|---|---|
| 联系人 | **全量**（含离线、含未分组） | 两个页签都能搜到所有人 |
| 讨论组 | 组名、成员名 | |
| 聊天记录 | 消息正文关键词 | 按会话聚合："与张三的聊天 · 3 条相关"，点击 → 进入会话并定位高亮该条 |
| 文件 | 收/发过的文件名 | 结果项可「打开」或「定位到聊天」 |

中文按字匹配、连续词组优先（基于本地索引，秒出结果；实现细节归技术设计）。

## 7. 关键流程

### 7.1 发起讨论组（参考微信：从单聊加人）

单聊头部 `⋯` →「添加成员」，或会话列表 `[+]` → 弹出**联系人多选器**（三级树 + 复选框 + 已选栏）→ 可选管理密码 → 确认创建。默认组名"张三、李四、王五的讨论组"，可改；若设置管理密码，成员管理群时需输入密码；若留空，仅创建 IP 可管理。

### 7.2 接收文件

对方发文件 → 我方聊天里出现文件卡片（「接收」「另存为」「拒绝」）→ 接收后显示进度 → 完成后卡片变为「打开 / 打开所在文件夹」。**≤20MB 的图片消息免确认**，直接以图片形式展示。会话不在前台时走通知 + 角标。

### 7.3 托盘与通知（点击直达会话）

- 新消息：Windows/Linux 托盘图标闪烁 + 系统通知横幅；macOS Dock 角标 + 通知中心。
- **点击通知或托盘 → 主窗弹出并直接定位到对应会话**（用户原话需求）。
- 托盘右键：打开主窗 / 全局免打扰开关 / 退出。

### 7.4 首次启动向导（决议 #22）

首次打开必经三步，防止全网一片"未命名"：① 昵称（必填，默认取系统用户名）→ ② 公司 / 部门 / 团队（选填，可跳过）→ ③ 文件保存目录确认 → 进主界面。中途退出则下次启动重新进入向导。

### 7.5 给离线的人发消息

正常输入发送 → 气泡时钟态 + 会话内一次性灰字提示"对方当前离线，将在其上线后自动送达"→ 对方上线后自动补发，气泡转 ✓。

## 8. 设置页

独立设置窗口（640×480，微信式），左侧分组导航，右侧表单。覆盖你点名的全部项 + 我补的必要项（标 ☆）：

| 分组 | 设置项 |
|---|---|
| 我的资料 | 头像（20 个亲和动物 emoji 图标 + 背景色模板，保留昵称首字模式）、昵称、公司、部门、团队 |
| 通用 | 开机自启（默认开）；☆ 关闭主窗口行为：最小化到托盘（默认）/退出；☆ 字体缩放 100%/110%/125%（办公场景护眼）；语言（置灰，远期） |
| 消息与通知 | 新消息系统通知开关；☆ 通知是否显示消息内容（隐私遮罩）；☆ 提示音（**默认关**，P1 使用系统通知声音开关，三款自定义试听留 v1.0 打磨，决议 #4）；☆ 发送键 Enter / Ctrl+Enter 互换 |
| 文件与存储 | 文件保存位置（路径 + 浏览 + ☆"每次询问"开关）；图片自动接收开关（≤20MB）；**聊天记录存储位置**（更换时自动迁移旧数据）；☆ 空间占用展示 + 清理图片缓存；**导出聊天记录**（选会话 + 时间范围 → HTML / TXT 阅读格式，或**迁移备份包**）；**导入聊天记录**（选备份包 → 预览会话/条数 → 合并导入：旧机器的"我"自动映射为本机、按消息 ID 去重，换电脑不丢数据）；☆ 清空所有聊天记录（双重确认） |
| 网络 | **IP 范围设置**（跨网段 CIDR 列表增删 + 「立即扫描」+ 上次扫描发现数）；☆ 手动添加节点 IP |
| 高级 | 监听网卡（默认全部）；UDP/TCP 端口（改后提示"全员需一致"并重启生效）；☆ 导出诊断日志；☆ 清理 90 天未上线联系人 |
| 快捷键 | 截图（默认 Ctrl/Cmd+Alt+A）、显示/隐藏主窗；☆ 截图时隐藏茶话间窗口（开关，决议 #22）；☆ 冲突检测红字提示 |
| 关于 | 版本号；☆ 发现内网更高版本时提示（开关）；开源许可 |
